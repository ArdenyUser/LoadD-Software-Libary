VISIX Project

We are proud to annouce the Visix project, a compiler to change OS developement! Also it has plenty other uses. Visix works like this:

Code > Converts to ASM or C# > Compiles

.VIS stands for: Vision Involved Scripting.

ALSO, we call the system Visix uses to Uncompilable Script to Compilable Script (USCS) Compiler. These USCS compilers are easier to make, and easy to use! Please wait until the beta is released!
